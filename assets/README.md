# Assets 디렉토리

이 디렉토리에 다음 아이콘 파일들을 추가할 수 있습니다:

- `icon.png` - 앱 아이콘 (권장 크기: 512x512 또는 256x256)
- `tray-icon.png` - 트레이 아이콘 (권장 크기: 16x16 또는 32x32, macOS는 템플릿 이미지 권장)
- `notification-icon.png` - 알림 아이콘 (권장 크기: 64x64)

## 아이콘 없이도 작동

아이콘 파일이 없어도 앱은 정상적으로 작동합니다. 아이콘을 추가하면 더 나은 사용자 경험을 제공할 수 있습니다.

## 아이콘 생성 도구

온라인 아이콘 생성 도구:
- [Favicon Generator](https://www.favicon-generator.org/)
- [Icon Generator](https://icon.kitchen/)

또는 이미지를 아이콘으로 변환:
- macOS: `sips -s format png input.jpg --out icon.png`
- 온라인: [CloudConvert](https://cloudconvert.com/)
